Dari Surabaya, Membangun Dunia

Halo! Saya Bima Syahputra, seorang Game Developer berbasis di Surabaya, Indonesia. Perjalanan saya dimulai dari porject hobby kecil Unity di bangku SMK.

Saat ini aktif mengembangkan proyek indie sambil terbuka untuk kolaborasi dan freelance project.
